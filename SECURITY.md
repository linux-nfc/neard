# Security Policy

## Reporting a Vulnerability

If you would liek to report a public issue, please send a bug report to the
mailing list:
    oe-linux-nfc@lists.linux.dev
or submit new issue on GitHub:
    https://github.com/linux-nfc/neard/issues

If you are dealing with a not-yet released or urgent issue, please submit
private vulnerability report via GitHub "Security and quality" tab:
    https://github.com/linux-nfc/neard/security
